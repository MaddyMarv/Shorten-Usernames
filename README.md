### [Shorten Usernames](https://www.nexusmods.com/warhammer40kdarktide/mods/641)
**The short version:** Shortens super long usernames in the HUD so they stop blocking your screen.

**The details:** Standalone mod that fixes massive usernames taking up half your screen. You can set a max character limit for character names and account names. It lets you shorten your own name and your teammates' names. You can pick how it cuts off the name, like adding dots or dashes at the end. You can also set it to only work during gameplay so you still see full names in the hub.
